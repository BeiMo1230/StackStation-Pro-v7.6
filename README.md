    StackStation Pro v7.6 全栈工作站使用手册

     StackStation 是一个轻量级、响应式的全栈文件管理与图床系统。它集成了资源分发、临时图床、数据监控及自动化运维功能，专为快速部署和高效管理而设计。
   一、 系统核心特点
   1. 响应式极致 UI 设计
    ●双端适配：采用 Tailwind CSS 驱动，完美适配 PC 宽屏、平板及手机端。
    ●沉浸式体验：集成动态时钟、运行时间计数器及磨砂玻璃（Backdrop Blur）视觉效果。
    ●交互动画：页面切换拥有平滑的缩放效果，上传进度条具备物理弹性反馈。
   2. 强化的文件处理能力
    ●全中文支持：底层采用 UTF-8 编码强化方案，完美支持中文文件名上传、显示与下载，彻底告别乱码。
    ●智能图标系统：资源中心根据文件后缀（PDF, ZIP, MP4, DOC 等）自动匹配特定的彩色图标。
    ●高速下载：支持文件直链复制与物理下载，通过 rawurlencode 确保跨浏览器下载兼容性。
   3. 自动化运维与安全
    ●图片自动清理引擎：管理员可自定义清理周期。系统自动遍历所有游客子目录，物理删除过期图片，节省磁盘空间。
    ●安全认证：采用 Session 隔离与管理员授权机制，核心管理功能（如主资源上传、配置修改）仅限授权用户。
    ●访客隔离：普通访客上传的图片存储在独立的 UID 目录下，互不干扰，保护隐私。
   二、 功能模块说明
   1. 资源中心
    ●(Files)用途：用于存储和展示长期资源（如软件安装包、文档、归档文件）。
    ●操作：点击资源卡片可弹出详情页，支持“复制下载直链”或“直接下载”。
   2. 图库服务 
    ●(Images)用途：临时图床功能。
    ●操作：点击“上传照片”即可将图片同步至云端。上传后可直接预览，并支持一键复制预览链接用于外部引用。
   3. 数据看板 (Dashboard) - 仅管理员
    ●用途：实时监控站点运营数据。
    ●内容：显示总访客数（基于独立会话）和文件累计下载次数。
   4. 后台管理 (Manager) - 仅管理员
    ●清理设置：设置图片存留的小时数（如设置 24 小时，则上传满 24 小时的图片将被自动物理删除）。
    ●里程碑同步：设置系统“启动时间点”，用于计算页面下方的“稳定运行时间”。
    ●资源监控：直接删除服务器上的冗余资源或图库图片。
   三、 操作指南
   1. 普通用户 (Guest)
    ●浏览与下载：直接进入“资源中心”查看公开资源并下载。
    ●临时传图：进入“图库服务”，点击上传。图片将保留至管理员设定的清理时限。
    ●获取 UID：页面右上角显示当前访客的唯一标识（UID）。
   2. 管理员 (Admin)
    ●登录系统：点击侧边栏下方的“管理员登录”，输入预设 ID 和 KEY。
    ●上传主资源：登录后，侧边栏会出现“上传主资源”按钮。点击后在底部中间会弹出实时进度条，显示同步状态。
    ●运维管理：定期在“管理中心”检查资源使用情况。通过修改“清理周期”来平衡服务器空间。
    ●退出系统：操作完成后点击“退出管理系统”以撤销管理权限。
   四、 技术规格
    ●后端：PHP 7.0+ (原生驱动，无需数据库)
    ●前端：HTML5, JavaScript (ES6),Tailwind CSS
    ●资产：Font-Awesome 6.4 (图标库)
    ●存储：本地 JSON 状态持久化 + 物理文件映射
